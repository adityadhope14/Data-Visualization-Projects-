# 📊 Data Visualization Projects

A collection of Python-based Data Analysis and Data Visualization projects developed using Jupyter Notebook.

## 🚀 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Projects Included

### 1. Exam Marks Statistics
- Load and analyze student marks
- Calculate mean, median, and standard deviation
- Identify above-average performers
- Generate statistical insights

## 📈 Features

- Data Cleaning
- Statistical Analysis
- Exploratory Data Analysis (EDA)
- Data Visualization
- Performance Analysis

## 🎯 Learning Objectives

This repository was created to strengthen:
- Data Analysis skills
- Python programming
- Statistical concepts
- Data Visualization techniques
- Real-world project development

## 🛠️ How to Run

1. Clone the repository

```bash
git clone https://github.com/adityadhope14/Data-Visualization-Projects.git
```

2. Navigate to the project directory

```bash
cd Data-Visualization-Projects
```

3. Launch Jupyter Notebook

```bash
jupyter notebook
```
